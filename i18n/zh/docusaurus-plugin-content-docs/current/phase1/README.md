# 阶段 1：Linux 与 Bash

> ⏱️ **预计时间：** 2-3 周

在 Learn to Cloud，我们主张尽早开始动手实践。因此，我们为你准备了一个夺旗赛（CTF）实验，供你练习 Linux 和 Bash 技能。在进入实验之前，你需要花时间了解并完成环境搭建。别担心 —— 在此过程中，我们将提供实用建议、文档链接、参考资源等，助你顺利完成。

---

## 🎯 学习目标

在该阶段结束时，你将完成：

- 在你的计算机上搭建好适用于后续所有实验的环境
- 学习版本控制的基础知识
- 熟悉云端命令行工具（CLI）
- 理解基础架构即代码（Infrastructure as Code）的概念
- 构建并访问实验环境
- 完成七个 CTF 挑战任务，以测试你的 Linux 技能

---

## 📋 准备工作

- Mac OS 或基于 Ubuntu 的计算机。如果你使用的是 Windows，则需要使用 WSL。
- 加入 [Learn to Cloud Discord 社区](https://discord.gg/Qymw28nQX6)
