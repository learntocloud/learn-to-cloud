# Linux 命令行 CTF 挑战

欢迎各位 CTF 参赛者！本指南将引导你完成本次 Linux 命令行夺旗赛（Capture The Flag）中的挑战任务。每个挑战都旨在测试你 Linux 技能的不同方面。

## 重要提示

当你完成每个挑战时，我们鼓励你记下任何你觉得有挑战性的概念或命令，并稍后重新学习。此外，我们诚恳地请求你**不要在社交媒体帖子中分享解决方案**。我们旨在杜绝“复制粘贴”式学习，并请求你在这方面的支持。相反，请在宣布完成实验的同时，分享你的思考和独特见解。

## 如何开始

本次 CTF 挑战要求你将实验环境从本地计算机部署到你首选的云服务提供商。所有过程都通过我们提供的文件实现了自动化。我们选择这种方式而不是托管实验环境，因为这将为你提供最接近真实世界的体验。

请首先访问该 [GitHub 仓库](https://github.com/learntocloud/ltc-linux-challenge)。
