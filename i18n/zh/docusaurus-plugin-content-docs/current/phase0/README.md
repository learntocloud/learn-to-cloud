# 阶段 0：从零开始

> ⏱️ **预计时间：** 3-4 周

如果你完全没有技术背景，但想进入云工程（Cloud Engineering）领域，那么这一阶段正是为你准备的。请花点时间掌握这些基础概念——深刻理解它们将使后续的学习变得更加轻松。

:::tip 已经有技术背景了？
如果你已经在技术领域工作并想转向云方向，请直接跳到 [阶段 1：Linux 与 Bash](../phase1/README.md)。
:::

---

## 🎯 学习目标

在该阶段结束时，你将：

- 理解 IT 和云计算的基础知识
- 从宏观层面了解 Linux、网络、编程和 DevOps 分别是什么
- 明确自己是否致力于追求云工程领域的职业生涯
- 为在这个不断发展的领域中持续学习做好准备

## 🎓 认证 (可选)

**对于进入技术领域的人来说，行业标准认证是 [CompTIA A+](https://www.comptia.org/certifications/a)。**

| 问题 | 回答 |
|----------|--------|
| 这是必须的吗？ | 否 |
| 这有帮助吗？ | 取决于你的学习方式 |
| 我应该考吗？ | 查看你所在地区的岗位需求——如果他们有要求，可以考虑考取 |

## 💼 入门级岗位

从支持性岗位（Support Role）开始是进入技术行业的一个极佳方式。这些职位通常对新人非常友好，并且可能提供学习材料和认证考试报销。
